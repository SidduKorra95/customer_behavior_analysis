
# 📊 Customer Behavior Data Analyst Project

## 🔍 Overview

This project demonstrates an end-to-end data analytics workflow, starting from database loading to delivering business insights through dashboards and reports. It highlights skills in Python, SQL, data cleaning, exploratory data analysis (EDA), and data visualization.

---

## 📁 Dataset

* The dataset is stored in a **MySQL database**
* Contains structured data used for analysis (e.g., customer behavior / sales / transactions)
* Data includes multiple tables and relevant attributes for analysis

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **MySQL Workbench** (for database management & SQL queries)
* **Power BI** (for dashboard creation)
* **Gamma** (for presentation slides)
* **Jupyter Notebook** (for analysis and EDA)

---

## ⚙️ Project Workflow

### 1. Data Loading

* Connected Python to MySQL database using SQLAlchemy
* Imported data into Jupyter Notebook for analysis

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Corrected data types
* Filtered inconsistent or invalid data

### 3. Exploratory Data Analysis (EDA)

* Analyzed distributions and patterns
* Identified trends and relationships
* Created visualizations for better understanding

### 4. SQL Analysis

* Wrote SQL queries in MySQL Workbench
* Performed aggregations, joins, and filtering
* Extracted insights directly from the database

### 5. Dashboard Creation

* Built an interactive dashboard in Power BI
* Included KPIs, charts, and filters for insights

### 6. Reporting & Presentation

* Created a detailed report summarizing findings
* Designed a professional presentation using Gamma

## ▶️ How to Run

1. **Clone the repository**

   ```bash
   git clone <https://github.com/SidduKorra95/customer_behavior_analysis.git>
   cd <customer_behavior_analysis>
   ```

2. **Install required libraries**

   ```bash
   pip install pandas numpy matplotlib seaborn sqlalchemy pymysql
   ```

3. **Set up MySQL connection**

   * Update credentials in the Python script:

   ```python
   username = "root"
   password = ""
   host = "localhost"
   port = "3306"
   database = "customer_behavior"
   ```

4. **Run Jupyter Notebook**

   ```bash
   jupyter notebook
   ```

5. **Open Power BI file**

   * Load the `customer_behavior_dashboard.pbix` file to view the dashboard
